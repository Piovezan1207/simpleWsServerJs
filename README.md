
## Simple websocket server example


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Piovezan1207/simpleWsServerJs
```

Entre no diretório do projeto

```bash
  cd simpleWsServerJs
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  node index.js
```
